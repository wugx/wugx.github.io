Compare tow tables of data in Hive

对比两张Hive表的数据是否一致

测试过程：

创建 tmp.wugx_table_1 表

create table tmp.wugx_table_1 (id string, name string) stored as orc ;

创建 tmp.wugx_table_2 表

create table tmp.wugx_table_2 (id string, name string) stored as orc ;

写入tmp.wugx_table_1 表数据

insert overwrite table tmp.wugx_table_1 values (‘1’,’ming’),(‘2’,’li’),(‘3’,’bai’);

写入tmp.wugx_table_2 表数据

insert overwrite table tmp.wugx_table_2 values (‘4’,’xiao’),(‘2’,’li’),(‘3’,’bai’);

查询结果：

第3列第1行数据代表此行数据仅在表1中存在，第2行数据表明仅在表2中存在
